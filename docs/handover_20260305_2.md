**方針:**
- `get_indicator_definitions(codes)` で `indicator_code → indicator_name` の辞書を取得

### 実装パターン（page_data_view.py を参考に）

各ページでの実装パターンは `page_data_view.py` に倣う:

```python
# 1. インポート追加
from src.data.indicator_loader import get_indicator_definitions

# 2. 状態変数に追加
code_to_name: dict[str, str] = {}

# 3. データロード時に取得
defs = get_indicator_definitions(df.columns.tolist())
code_to_name = dict(zip(defs["indicator_code"], defs["indicator_name"]))

# 4. 表示時に変換（内部処理はコードのまま）
label = code_to_name.get(code, code)  # フォールバックあり
```
