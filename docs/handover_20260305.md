## 次セッションで対応すべき問題

### ヘルプパネルの内容がグレーボックスで表示される

**症状:**
ヘルプパネルを展開すると「操作手順」「表示される結果」の2列がグレーの空ボックスになり、テキストが見えない。

**原因の仮説:**
`components/help_panel.py` の `content` 部分で、`ft.Column` 内の `ft.Text` が `expand=True` の `ft.Row` の中に置かれているが、Flet 0.81.0 では `ft.Text` に `expand=True` を指定しても高さが確定しないケースがある。または `ft.Column` の `visible=False` → `True` 切替時に再描画が正しく行われていない可能性。

**調査・修正すべきファイル:**
- `components/help_panel.py`（主要原因）

**修正の方向性:**
1. `ft.Text` の `expand=True` を削除し、`ft.Container` でラップして幅を確定させる
2. または `ft.Column` の `visible` 切替を `content` の `height` 切替（0 ↔ auto）に変える
3. または `ft.Row` の代わりに `ft.ResponsiveRow` を使う
